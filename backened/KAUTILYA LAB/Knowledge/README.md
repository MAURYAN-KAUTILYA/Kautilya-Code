# Knowledge

This folder is the doctrine shelf for Kautilya Code.

## What It Contains

- `code/` - system prompts, architecture rules, UX guidance, and adversarial doctrine
- `Global/` - voice, reasoning, rules, and quality standards
- `memory/` - durable JSON memory snapshots
- `sessions/` - session history and resumable context
- `checkpoints/` - saved checkpoints for interrupted work

## Why It Matters

The app is opinionated because the knowledge base is opinionated. These files shape tone, behavior, guardrails, and the product's internal discipline.
