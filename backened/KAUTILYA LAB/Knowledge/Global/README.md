# Global Knowledge

This folder defines the global voice and standards for the system.

## Files

- `chanakya.md` - strategic voice and identity
- `personality.md` - response tone and persona
- `prompt-patterns.md` - reusable prompt structures
- `quality-bar.md` - output quality expectations
- `reasoning.md` - reasoning discipline and decision style
- `rules.md` - general behavioral rules

## Why It Matters

These files give Kautilya a consistent personality across the whole product. They keep the system recognizably itself, no matter which route or backend path is active.
