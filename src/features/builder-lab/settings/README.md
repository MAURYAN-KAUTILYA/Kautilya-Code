# Builder Settings

Settings modules for the builder live here.

## What Lives Here

- `SettingsSheet.tsx` - the reusable settings container
- `registry.tsx` - feature registry for settings modules
- `types.ts` - shared settings types
- `theme-builder/` - the theme customization feature

## Why It Matters

This folder keeps the builder extensible. New settings can be added as features instead of becoming one-off screens.
