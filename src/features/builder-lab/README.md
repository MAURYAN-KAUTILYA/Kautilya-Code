# Builder Lab

This is the flagship surface of Kautilya Code.

## What Lives Here

- `BuilderShell` - the main workspace coordinator
- `Aipanel` - command, model, and workflow controls
- `editor` - code editing and workspace presentation
- `preview` - preview and console rendering
- `SketchBoard` - draggable notes and visual planning

## Why It Matters

The builder is where the product feels alive. Prompt, file tree, terminal, preview, approval, and sketching all stay in the same frame so the work feels continuous.

## Note

The `.tsx` files are the primary source. The `.js` files are mirrors that keep the same surface available for compatibility.
